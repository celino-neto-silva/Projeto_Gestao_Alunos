#Projeto Gestão de Alunos
-------------------------

#Descrição:

O presente projeto tem como objetivo desenvolver um sistema de gesão de alunos utilizando Spring Boot e Spring Data Repository.

O núcleo do negócio é gerir alunos com determinados recursos de gestão disponibilizados para o utilizador. Vamos abaixo ver esses recursos que também podem ser chamados de Requisitos Funcionais/Funcionalidades.

#Requisitos Funcionais / Funcionalidades:

Adicionar aluno
Remover aluno
Listar alunos

#Requisitos Técnicos:

Geração do projeto utilizando Spring Initializer
Adicionar depedências:
    Spring Web
    Spring Boot Dev Tools
    Lombok
    Thymeleaf
    JPA
    H2 database
    
Utilizar maven
JDK 21
Configurar base de dados no ApplicationProperties
Ligação com base de dados ("Spring Data JPA")
Utilização de componentes Spring Boot
    Models
    Repository
    Controller
    View
Utilização de Thymeleaf para as Views
Utilização de CSS
Utilização de JavaScript
